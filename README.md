# Dartbord Teller 🎯

Zelfstandige (offline) Nederlandse darts-teller (X01) in één HTML-bestand:

- Klikbaar dartbord + score-engine (301/501/701, dubbel-uit, checkout-advies)
- Nederlandse voorlees-stem (kiest automatisch de meest natuurlijke stem)
- Leuke visuals (confetti bij 180 en checkout)
- **Camera-modus** met 4-punts perspectief-kalibratie en experimentele automatische pijl-herkenning

**Live:** https://tmvsb5797f-hub.github.io/dartbord-teller/

> De camera werkt alleen via https (zoals deze Pages-URL), niet als lokaal bestand.
